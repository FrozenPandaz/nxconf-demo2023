2. Import API
3. `pnpm i @nx/node -D`
4. `nx g app api`
5. `rm -rf api/src`
6. `cp -r ../nxconf-demo2023-api/src api`
7. `pnpm i fastify fastify-plugins @fastify/sensible @fastify/autoload`
8. Update Transaction type
